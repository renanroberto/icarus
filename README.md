# Projeto Icarus

Um boilerplate rápido e flexível para e-commerce

## Como usar

...

## Features

- Configuração fácil
- Super rápido!
- Hot Reload (Vtex Local)
- Cross Browser configurável (*default: Duas ultimas versões de cada browser e browsers com mais de 1% de uso global*)
- React

## Stack

- Webpack
- Babel
- React
- PostCSS
